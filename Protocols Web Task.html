HTTP is a protocal in which servers across the internet follow when files are being transmitted. It is the most type of protocal and was used in all of the first created web pages. Watch the Video below to find out more about Hypertext Transfer Protocol.



Hypertext Transfer Protocol Secure (HTTPs)

HTTPS is the better option out of HTTP and HTTPS as it is more secure. The HTTPS protocals allows files to be transmitted through a safe encrypted connection. This therefore prevents the likelyhood of any files being stolen between the transfer process between server and browser. Transport Layer Security and Secure Sockets Layer (TLS/SSL)

TSL also known as SSL is a protocal that encrypts data that is sent via the internet. It checks and verifies the users and therefore provides end-to-end security to ensure that the private and sensitive information that may be sent across is unable to be intercepted by hackers.
